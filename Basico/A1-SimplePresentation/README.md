  - I + am + Complement
  - He/She/It + is + Complement
  - You/We/They + are + Complement
- **Negative Sentences**
  - I + am not + Complement
  - He/She/It + is not / isn‚Äôt + Complement
  - You/We/They + are not / aren‚Äôt + Complement
- **Interrogative Sentences**
  - Am + I + Complement + ?
  - Is + He/she/It + Complement + ?
  - Are + You/We/They + Complement + ?
